\section{Summary of Contributions}
% ─────────────────────────────────────────────────────────────
This thesis designed, implemented, and evaluated the \textbf{Serverless Spark Framework
for Real-Time IoT Applications} with an integrated Two-Phase Hierarchical RL Optimizer
and OpenFaaS Serverless Orchestration. Four interlinked contributions were made:

\begin{enumerate}
  \item \textbf{Serverless IoT Pipeline:} A containerised five-layer architecture
        (MQTT + Kafka + Spark + Redis + OpenFaaS) achieving 310~ms end-to-end latency
        at 6.4\% total CPU usage.
  \item \textbf{PPO Resource Optimizer:} A Gymnasium-based simulator enabling offline
        PPO training in $\sim$8 min, outperforming all baselines on cost, latency, and
        throughput.
  \item \textbf{Two-Phase Hierarchical RL:} A TD3 Meta-Controller dynamically shifts
        reward weights $(\alpha,\beta,\gamma)$ between cost-minimisation and
        latency-priority without retraining the PPO agent.
  \item \textbf{OpenFaaS Deployment:} Scale-to-zero RL inference achieving \$0.00 idle
        cost (vs.\ \$11.25 traditional), with independent model updates and HTTP-based
        Spark decoupling.
\end{enumerate}

\section{Final Remarks}
% ─────────────────────────────────────────────────────────────
The Serverless Spark IoT Framework demonstrates that adaptive intelligence can be
embedded as a first-class architectural component in distributed cloud systems. By
achieving simultaneous improvements in latency ($-$51.4\%), cost ($-$36.5\%),
SLA compliance (100\%), and throughput (+2.2$\times$), this work establishes a
blueprint for self-adaptive, cost-efficient serverless analytics platforms for
the Internet of Things.
